# Photo in Word

Скрипт для размещения фотографий в Word-документе с альбомной ориентацией.

## Возможности

- Поддержка популярных форматов: JPG, PNG, GIF, BMP, TIFF, WebP, HEIC (iPhone)
- Равномерное распределение фото на странице в виде сетки
- Альбомная ориентация страниц
- Нумерация страниц (кроме первой)
- Настраиваемые отступы между фотографиями (по умолчанию 0.5 см)
- Автоматическая подгонка размера фото под страницу с сохранением пропорций

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Сборка в .exe

Запустите `build_exe.bat` — будет создан файл `dist\PhotoInWord.exe`, который можно запускать без установки Python.

## Использование

1. Укажите путь к папке с фотографиями
2. Введите количество фото на страницу (1–12)
3. При необходимости укажите отступы между фото в см (по умолчанию 0.5)

Документ `photos.docx` будет сохранён в ту же папку, где находятся фотографии.
